# 기여 가이드

## 개발 환경

- Node.js 18 이상
- Python 3.9 이상 (tiktoken, LangChain 사용 시)

## 실행 방법

```bash
git clone https://github.com/0727n1122-beep/graduationproject.git
cd graduationproject
node src/demo.mjs
```

브라우저에서 http://localhost:3000 접속

## 브랜치 전략

- `main` — 안정 버전
- `dev` — 개발 중인 기능
- `feature/*` — 개별 기능 브랜치
