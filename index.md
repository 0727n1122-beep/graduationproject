# 💰 프롬프트 자동 최적화 기반 LLM API 비용 실시간 비교 웹 서비스

> AI 기반 LLM API 비용 사전 비교 플랫폼
> 

| 항목 | 내용 |
| --- | --- |
| **프로젝트명** | 프롬프트 자동 최적화 기반 LLM API 비용 실시간 비교 웹 서비스 |
| **한줄 설명** | 사용자가 입력한 프롬프트를 비용적으로 최적화한 뒤 ChatGPT, Gemini 등 각 LLM의 예상 API 비용을 한눈에 비교해주는 서비스 |
| **프로젝트 키워드** | LLM 비용 비교, API 토큰 계산, 프롬프트 최적화 |
| **무엇을 만드는가** | 사용자가 프롬프트를 입력하면, 입력된 프롬프트를 자동 최적화하고, 각 AI 모델(GPT-4, Gemini, Claude 등)의 예상 비용을 실시간으로 계산하고 비교하는 웹 서비스 |
| **타겟 고객** | AI API를 사용하는 학생, 개발자, 스타트업, 기획자 등 비용에 민감한 사용자 |

## 💡 아이디어를 떠올린 이유?

- ChatGPT, Gemini, Claude 등 AI 모델이 쏟아지면서 개발시 어떤 모델을 쓸지 고민하게 됨.
- 성능도 중요하지만 비용도 매우 중요함.
- api형 ai 모델들은 토큰 단위로 과금
    - 그런데 각 회사마다 요금 체계가 달라서 실제로 내 프롬프트를 돌리면 **얼마가 나오는지 미리 알기가 어려움.**

## 📚 용어 설명

- **토큰(Token)** : AI가 텍스트를 처리하는 단위. 대략 한글 1글자 = 2~3토큰, 영어 1단어 = 1~2토큰. AI 요금은 '토큰 수'로 계산됨.
- **LLM(Large Language Model)** : GPT-4, Gemini, Claude 같은 대형 언어 AI 모델의 총칭.
- **프롬프트(Prompt)** : AI에게 보내는 질문이나 명령문. '이 글을 요약해줘' 같은 입력값.

## 🛠 핵심 기능

- 프롬프트 입력 → 프롬프트를 비용적으로 최적화
- 최적화된 프롬프트로 각 LLM 모델의 예상 토큰 수 및 비용 자동 계산
- "Gemini: 800원 / GPT-4: 1,200원 / Claude: 950원" 형태로 비교 카드 UI 제공
- LLM별 가격 DB 자동 업데이트 기능

## 👀 부가 기능

- 월간 사용량 대시보드: API 호출 내역을 누적하여 "이번달 GPT-4에 얼마 썼어요, 이렇게 바꾸면 OO만원 아낄 수 있어요" 를 보여주는 리포트 생성
- **프롬프트 템플릿 저장소:** 사용자가 자주 쓰는 프롬프트를 저장하고, 저장된 프롬프트의 월간 예상 비용을 미리 계산해주는 기능
- 예산 알림: 사용자가 설정해둔 월별 예산을 초과하는 경우 Slack이나 이메일로 알림 발송.
